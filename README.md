This project focuses on building and evaluating regression models to predict the total transaction value using the Online Retail dataset. The dataset was first cleaned by removing missing values and filtering out invalid entries such as negative quantities and prices. Feature engineering was performed to enhance model performance, including converting the InvoiceDate into a datetime format and extracting useful time-based features such as year, month, day, and hour. A new target variable, TotalPrice, was created by multiplying quantity and price, while categorical data like country was encoded into numerical form. Three regression models—Linear Regression, Ridge Regression, and Lasso Regression—were developed and trained on the processed dataset. The models were evaluated using performance metrics such as R-squared, Mean Squared Error (MSE), Root Mean Squared Error (RMSE), and cross-validation scores to assess their predictive accuracy and generalization ability. Among the models, Lasso Regression performed slightly better, achieving the lowest error and highest R-squared value.Overall, the results indicate that while the models provide reasonable predictions, there is scope for improvement through advanced feature engineering and more complex modeling techniques.
